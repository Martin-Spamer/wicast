# WiCast

WiCast Wireless MultiCasting

## Usage

### Clone to your local

### Build with Maven

### Add dependency to your project

	<dependency>
		<groupId>net.wicast</groupId>
		<artifactId>wicast</artifactId>
		<packaging>jar</packaging>
		<version>1.0-SNAPSHOT</version>
	</dependency>
